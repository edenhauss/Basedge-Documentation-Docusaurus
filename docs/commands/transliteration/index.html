Транслит

Транслит — это наиболее удобный для чтения текстовый формат для транслитерации слов. Он позволяет сохранить все особенности написания и чтения, сохранить текст в неизменном виде в любое время. Кроме того, его можно быстро редактировать и использовать при переводе текста.
